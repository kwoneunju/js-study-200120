## 스터디 진행
1. 코드 실행 순서 작성(한글로)
2. 구글링 전, 코드 작성 시도 (2번 이상 시도)
3. 구글링(적어도 2개 이상의 방법 찾기)
4. 구글링한 코드 **이해하고 자기걸로 만들기**
5. 구글링 전에 작성한 코드 실행 순서와 구글링으로 찾은 방법 서로 비교 및 수정
6. 구글링한 코드를 자신의 것으로 만들면서 알게된 jQuery 정리(+ 질문도)
